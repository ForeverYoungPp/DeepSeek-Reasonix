feasibility legend

full  直接可移植到 Ink。要么是 0.12.x 已实现，要么是 Box / Text / 颜色组合就能做。光标 blink、spinner、流式 token、card 边框、accent bar、pill 高亮、stacked 进度条都属于此类。

partial · ~85%  骨架可做但损失"丝滑感"。滑入动画→突现 + 2 帧色阶；backdrop blur→全屏 dim 重绘；box-shadow 发光→双层边框 + accent 色；hover→只在 focus 时变化（TUI 没鼠标 hover）。

new feature  Reasonix 现在没这能力，需要写代码。restoreCheckpoint（替代被毙的 auto-commit）/ workspaceCwd 确认 modal / subagent inline embed 都属此类——需先评估再做。

drop  终端单元格物理限制无法做的。亚像素抗锯齿圆角、per-pixel gradient 渐变背景、mouse hover 跟随。本设计里没有任何 surface 完全依赖这一档——所有视觉都有降级路径。
